# Agreements

Open-source contract templates used by **Charper Bonaroo B.V.** for client engagements and partnerships. All documents are provided as Markdown source files (and PDF exports where applicable) so they can be versioned, diffed, and adapted easily.

## What's in this repo

The repository contains the following agreement templates, available in both Dutch (NL) and English (EN):

| Document | NL | EN |
|---|---|---|
| General Terms & Conditions | `algemene_voorwaarden.md` | `terms_and_conditions.md` |
| Non-Disclosure Agreement | `geheimhoudingsovereenkomst.md` | `non_disclosure_agreement_(en).md` |
| Assignment Agreement | `overeenkomst_van_opdracht.md` | `assignment_agreement.md` |
| IP Transfer Deed | `akte_van_overdracht.md` | — |
| Data Processing Agreement | — | `data_processing_agreement.md` |

PDF exports are included for the General Terms & Conditions (`algemene_voorwaarden.pdf`, `terms_and_conditions.pdf`) and the NDA (`geheimhoudingsovereenkomst.pdf`).

## Bonaroo policies

- The Algemene Voorwaarden (General Terms & Conditions) always apply to every engagement.
- NDAs are always symmetrical — both parties have equal obligations.
- All agreements include a fixed termination date.

## How to use

1. Pick the template you need and copy it.
2. Fill in the blanks (marked with underscores `______`) with the relevant party names, dates, amounts, and scope descriptions.
3. The Assignment Agreement and Overeenkomst van Opdracht reference the Algemene Voorwaarden by URL — make sure to include a copy when sending to the other party.
4. For IP transfers, use the Akte van Overdracht after the work described in an Assignment Agreement has been completed.
5. Attach a Data Processing Agreement whenever personal data will be processed as part of the engagement.

## Limitations

- **Not legal advice.** These templates are shared for reference and convenience. They are not a substitute for professional legal counsel. Have a qualified lawyer review any agreement before signing.
- **Dutch law.** All agreements are governed by Dutch law and refer disputes to Dutch courts. They may not be suitable (or enforceable) in other jurisdictions without modification.
- **Bonaroo-specific.** The templates are written from Bonaroo's perspective and contain Bonaroo's company details. If you reuse them for your own organisation, make sure to replace all references accordingly.
- **PDF exports may be outdated.** The Markdown files are the source of truth. The included PDFs may lag behind the latest Markdown edits.
- **Incomplete translations.** Not every document is available in both languages yet (e.g. the IP Transfer Deed is NL-only, the Data Processing Agreement is EN-only).
